# Описание
Скрипт detail_detector реализован на python, и предназначен для детектированя формы и размеров изделия по входящему растровому чертежу. Результатом работы скрипта является каталог out, с файлами verlay.png на котором цветом обводится форма изделия, а так-же выделяются его размеры, и result.json, в котором в формализованном текстовом формате описывается детектированное изделие.

### Установка
Для обеспечения работы скрипта, необходимо установить зависимости, командой:
```
pip install -r requirements.txt
```

Oпционально, для установки зависимостей можно установить виртуальное окужение, командой:
```
python -m venv venv
```
и зайти в него, командой зависящей от используемой операционной системы.

### Запуск
Для запуска скрипта используется интерпритатор python в который в качестве первого аргумента подаётся скрипт detail_detector.py, в качестве второго аргумента подаётся путь до растрового чертежа, например:
```
python detail_detector.py drawing.jpg
```

### Техническая аннотация
Скрипт разрабатывался, и тестировался в ОС Ubuntu 24.04.4 LTS, на Python 3.11
